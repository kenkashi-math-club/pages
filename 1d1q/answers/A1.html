問A1 解答  answer

解法1

\begin{align} &\int_{-1}^1 \left( x^4-\frac{5}{2}x^3+2x^2-11x+4\right) dx\\ =&\left[ \frac{1}{5}x^5 - \frac{5}{6}x^4 + \frac{2}{3}x^3 - \frac{11}{2}x^2 + 4x \right]_{-1}^1\\ =&\frac{1}{5}\{1^5-(-1)^5\} - \frac{5}{6}\{1^4-(-1)^4\} + \frac{2}{3}\{1^3-(-1)^3\} - \frac{11}{2}\{1^2-(-1)^2\} + 4\{1-(-1)\}\\ =&\frac{2}{5} - 0 + \frac{4}{3} - 0 + 8\\ =&\mathbf{\frac{146}{15}} \end{align}

別解

\(\displaystyle \int_{-a}^a x^{2n}dx = 2\int_0^a x^{2n}dx,\qquad \int_{-a}^a x^{2n-1} dx =0\: \text{（\(n\)は整数）}\) を利用すると， \begin{align} &\int_{-1}^1 \left( x^4-\frac{5}{2}x^3+2x^2-11x+4\right) dx\\ =& 2\int_0^1 (x^4+2x^2+4)dx\\ =&2\left[\frac{1}{5}x^5 + \frac{2}{3}x^3 + 4x \right]_0^1\\ =&2\left( \frac{1}{5}+\frac{2}{3} + 4\right)\\ =&\mathbf{\frac{146}{15}} \end{align}
